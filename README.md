# Junior-Data-Analyst
 
## Kirish

Ushbu portfolio, ma'lumotlar tahlili va vizualizatsiyasi bo'yicha asosiy ko'nikmalarni rivojlantirish uchun mo'ljallangan. Loyiha turli xil ma'lumotlar to'plamlarini tahlil qilish, statistik usullarni qo'llash va natijalarni grafik ko'rinishda taqdim etishni o'z ichiga oladi.

## Maqsad

- Ma'lumotlarni to'plash va tahlil qilish
- Vizualizatsiya vositalaridan foydalanish
- Hisobotlar tayyorlash va natijalarni taqdim etish
